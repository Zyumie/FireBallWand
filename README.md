# Le FireBallWand V2 est un plugin pour les Admins,

> Il permet de :
- Utiliser un Bâton de Feu surpuissant,
- Utiliser un FreezeWand sur des joueur en survie,
- Utiliser un KickArmor qui kick les joueur autour du porteur (n'étant pas Op). 

> Télécharger le FireBall V2 : https://modrinth.com/plugin/fireballwand/version/2
